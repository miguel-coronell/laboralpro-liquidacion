# LaboralPro | Calculadora de Liquidación de Contrato

Aplicación web que permite calcular la liquidación laboral de un trabajador de forma rápida y sencilla.

## Funciones

- Cálculo de salario pendiente
- Cálculo de cesantías
- Intereses de cesantías
- Prima de servicios
- Vacaciones
- Estimación de liquidación total

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

## Objetivo del proyecto

Este proyecto fue desarrollado como herramienta práctica para estimar la liquidación laboral de empleados de manera automática.
